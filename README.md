##Vote The Bowl - Web UI

Vote The Bowl is an easy to use cooking contest themed voting web application. By design it requires no account creation or personal data, apart from a name while voting (this is subject to change in the future as new features are added)
The UI is build using SvelteKit and Svelte-Flowbite.

### Env

You must have an .env file in the project directory for Production.

```sh
PUBLIC_RESULTS_REFRESH_DELAY=3000
```
